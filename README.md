# AI Smart Code Reviewer

AI Smart Code Reviewer is a demo application that analyzes source code and provides automated feedback such as **bugs, security issues, performance problems, and improvement suggestions**. The project demonstrates how AI can assist developers in performing **basic static analysis and automated code review**.

This project includes a **React frontend** and a **Python backend** that processes code input and returns structured analysis results.

---

## Features

- Automated code analysis
- Detects potential **bugs**
- Highlights **security issues**
- Identifies **performance problems**
- Suggests **code improvements**
- Displays results in a structured format
- Simple and interactive web interface
